# Libft + printf + printerror
Libft is a custom C library built from scratch as part of the 42 School curriculum. It reimplements core standard functions, along with a custom printf and a variant writing to stderr. Focused on memory management, data structures, and clean, reusable low-level code.

# Features
Reimplementation of core C standard library functions
Custom memory management utilities
String and character manipulation functions
Linked list implementation and helpers
Custom printf implementation
printf variant that writes to stderr

# Technologies
Language: C
Standard: C99 (or project-required standard)
Compiler: cc / gcc
Build system: Makefile

# Learning Goals
Deep understanding of how standard C functions work internally
Mastery of pointers and memory management
Writing modular and reusable code
Handling low-level I/O operations
